//Master directory of all programming assignments
Program1 - pairsofwords - Program to read in any # of files, process pairs of words contained therein, then sort and output a specified # of pairs by occurrence .
Program2 - dictsearch - Program to read words at a byte level, compare to a dictionary, determine where in the dictionary the word occurs if it does.
Program3 - readable - Program to recurse through directories and count the # of readable regular files found therein. Ignores symbolic links and files it cannot access.
Program4 - pipe: - Imitates the functionality of pipe() by forking processes
Program5 - diningPhilosophers - semaphore implementation of a solution to the dining philosophers problem
